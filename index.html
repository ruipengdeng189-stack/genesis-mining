Build Plan

Why It Works Without Paid Backend

Hub First The root page only handles navigation. Each game lives under games/.
Local Save Each game saves with localStorage, and the hub reads those values on the same domain.
Free Deployment You still deploy through the same GitHub -> Vercel auto-publish flow.
Expand Later Add a lightweight data layer later only when you truly need cross-device sync.
