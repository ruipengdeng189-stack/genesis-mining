## 1. 建模前提

这份表按**保守模型**计算：

- 只计入：
  - 初始资源
  - 章节通关奖励
  - 章节首通奖励
- **不计入**：
  - 任务奖励
  - 赛季奖励
  - 每日免费补给
  - 商店追赶包

所以文档里的白嫖路径是“保守下限”，真实玩家因为会拿到任务、赛季、每日补给，实际可支配资源通常会**更宽松一档**。

| 资源 | 数值 |
| --- | ---: |
| 金币 `Credits` | 1680 |
| 密钥位 `Key Bits` | 180 |
| 密码尘 `Cipher Dust` | 56 |

### 默认起始构筑

- 队长：`glintFox`
- 技能：`gridBurst`
- 模块：`prismTap` + `hardPatch`
- 起始构筑战力：`260`

### 核心构筑轮次定义

文档中的“**1 轮核心构筑**”指：

- 当前队长 +1 级
- 当前技能 +1 级
- 当前两张模块各 +1 级

当前版本下：

- 每轮核心构筑固定提升原始战力：`+76`
- 研究 `stabilityMesh` 每级再给整套构筑 `+4%`

## 5. 当前卡点设计

### A. `1-3` 首个软卡点

- **目标体验**：让玩家在第一个首领前学会：
  - 升当前主力构筑
  - 理解护盾目标
  - 认识“首补免费”和开局资源的重要性

#### 白嫖通过路径

- 推荐状态：
  - `2` 轮核心构筑
  - 或 `1` 轮核心构筑 + `stabilityMesh` `L2-L3`
- 资源参考：
  - `2` 轮核心构筑累计：`1407 / 163`
  - 玩家到 `1-2` 后保守资源：`2571 / 280 / 91`
- 结论：
  - **白嫖可稳过**
  - 这里不是强收费墙，而是第一波“升级教学”

#### 小氪通过路径

- 推荐购买：`starterPack`
- 对应增强：
  - 立即给 `2600 Credits / 220 Bits / 72 Dust`
  - 永久 `+1` 每日免费局
  - 每局开局额外能量更高
- 价值对位：
  - 约等于 **2 轮核心构筑 + 早期研究**
  - 直接覆盖 `1-3 / 2-1`

#### 中氪通过路径

- 不建议在这个点就上中氪
- 更合理的中氪入口是 `2-2 / 2-3`

---

### B. `2-2 / 2-3` 中期研究卡点

- **目标体验**：逼玩家从“只升卡”过渡到“卡牌 + 研究 + 商店补给”
- `2-3` 是第一道真正的中期资源压力点

#### 白嫖通过路径

- 推荐状态：
  - `4-5` 轮核心构筑
  - `stabilityMesh L4-L5`
  - `pulseBattery L2`
  - 领取任务 / 每日补给 / 赛季免费档位
- 保守资源参考：
  - 到 `2-2` 后保守资源：`5346 / 596 / 195`
  - `4` 轮核心构筑累计：`3812 / 396`
  - `stabilityMesh L4` 累计：`1399 / 102`
  - `pulseBattery L2` 累计：`371 / 27`
- 结论：
  - 单靠“只升卡”会开始发紧
  - 但白嫖只要把任务、每日免费补给和免费赛季领满，仍能过 `2-3`

#### 小氪通过路径

- 推荐购买：`starterPack`
- 通过方式：
  - 用礼包资源把核心构筑提到 `4-5` 轮
  - 早期研究补到 `stabilityMesh L4`
  - 靠永久开局增益稳过 `2-3`
- 设计意图：
  - 让小氪感受到“**省 2-3 天追赶时间**”，而不是直接跳过整个第二章

#### 中氪通过路径

- 推荐购买：`seasonPass`
- 对应增强：
  - 立即给 `4200 Credits / 320 Bits / 160 Dust / 180 Season XP`
  - 解锁高级赛季轨道
  - 研究价格额外 `-15%`
  - 第二章开始额外给开局能量
  - 第三章开始额外给中后段收益与开局容错
  - 最大能量额外 `+12`
- 价值对位：
  - 约等于 **3 轮核心构筑 + 稳定网格 L4**
  - 是专门对 `2-2 / 2-3` 研究卡点设计的中氪解法

---

### C. `3-3` 后段首领卡点

- **目标体验**：从“数值够不够”升级成“数值 + Boss 对策是否成型”
- 这里开始要求：
  - 构筑轮次更高
  - `stabilityMesh` 不再能缺席
  - 需要切到更适合首领的 Boss 构筑

#### 白嫖通过路径

- 推荐状态：
  - `7-8` 轮核心构筑
  - `stabilityMesh L5-L6`
  - `pulseBattery L3-L4`
  - 构筑换成 Boss 取向：
    - 队长优先 `novaEcho` / `wardenNine`
    - 模块优先 `traceMine` / `burstIndex`
    - 技能优先 `stasisField` 或 `colorHack`
- 保守资源参考：
  - 到 `3-2` 后保守资源：`11141 / 1253 / 413`
  - `7` 轮核心构筑累计：`10979 / 951`
  - `8` 轮核心构筑累计：`14976 / 1214`
- 结论：
  - 白嫖能过，但需要把任务 / 免费赛季 / 每日补给几乎都吃满
  - 这是“熟练玩家区间”的真正分水岭

#### 小氪通过路径

- 推荐购买：`starterPack + seasonPass`
- 通过方式：
  - `starterPack` 负责早期抢节奏
  - `seasonPass` 负责中期研究折扣 + 高级轨道补给 + 中后段开局/收益补强
- 设计效果：
  - 小氪玩家能在 `3-3` 前把资源差拉开一整轮
  - 但仍然需要切 Boss 构筑，不能无脑平推

#### 中氪通过路径

- 推荐购买：`breakerVault`
- 对应增强：
  - 立即给 `18000 Credits / 1350 Bits / 560 Dust`
  - 永久提升后段章节收益
  - 永久减轻首领反制
- 价值对位：
  - 约等于 **8 轮核心构筑 + 一组中高阶研究**
  - 是针对 `3-3 / 4-x` Boss 墙的专门付费解

---

### D. `4-2 / 4-3` 终盘卡点

- **目标体验**：必须完成“成长闭环”
- 这里不再只是等级门槛，而是：
  - 高轮次核心构筑
  - 中高等级研究
  - Boss 专项增益
  - 长周期资源经营

#### 白嫖通过路径

- 推荐状态：
  - `9-10` 轮核心构筑
  - `stabilityMesh L6+`
  - `pulseBattery L4+`
  - 免费赛季和任务尽量全吃
- 特点：
  - 可过，但时间成本很长
  - 这是终局长线留存区

#### 小氪通过路径

- 推荐购买：
  - `starterPack`
  - `seasonPass`
- 通过方式：
  - 保持日常经营，把 `4-2 / 4-3` 当作 1~2 周的终盘目标

#### 中氪通过路径

- 推荐购买：
  - `starterPack`
  - `seasonPass`
  - `breakerVault`
- 通过方式：
  - `breakerVault` 直接把后段 Boss 战从“硬拖”改成“可控”
  - 让中氪玩家感到“这单花得值”，但仍要完成构筑与研究

## 6. 三档付费与卡点映射

| 档位 | 推荐触发点 | 资源价值 | 永久增益 | 解决的核心问题 |
| --- | --- | --- | --- | --- |
| `starterPack` | `1-3 / 2-1` | 约等于 2 轮核心构筑 + 早期研究 | 每日免费局 +1、开局更稳 | 解决第一波升级焦虑 |
| `seasonPass` | `2-2 / 2-3` | 约等于 3 轮核心构筑 + 中期研究包 | 研究更便宜、中后段开局更稳、后段收益更顺 | 解决中期研究卡点与后段小氪续航 |
| `breakerVault` | `3-3 / 4-x` | 约等于 8 轮核心构筑 + 高阶研究包 | 首领反制减轻、Boss 战更可控、后段收益提高 | 解决后段 Boss 墙 |

## 7. 设计原则总结

当前版本的数值逻辑不是“硬性逼付费”，而是：

1. **白嫖能过**  
   只要任务、每日、免费赛季、研究和构筑都跟上，就能推进。

2. **小氪买时间**  
   小氪不直接跳章节，而是缩短 1~3 天的资源追赶周期。

3. **中氪买卡点解决方案**  
   中氪不是只买资源，而是买“当前卡点真正缺的那部分增强”：
   - 早期买开局稳定
   - 中期买研究效率
   - 后期买 Boss 对策

4. **高阶卡点必须保留操作价值**  
   就算买了 `breakerVault`，玩家也仍然需要切 Boss 构筑、合理放技能、先打护盾再打核心目标。

如果后续还要继续往下调，我建议下一轮优先看两件事：

- `2-3` 的真实失败率是否落在 `25%~40%`
- `3-3` 的付费前后通关率差是否控制在“明显感知到变强，但不会一键碾压”

## 8. 目标失败率区间

以下建议优先观察**首次尝试失败率**，并同时对照**3 次内通关率**：

- **白嫖**：不持有任何付费包，只吃任务、每日免费补给和免费赛季。
- **小氪**：只持有当前卡点对应的一档包。
- **中氪**：持有前序推荐包 + 当前卡点对应的一档包。

| 关卡 | 白嫖首次失败率 | 小氪首次失败率 | 中氪首次失败率 | 设计意图 |
| --- | --- | --- | --- | --- |
| `1-1` | `12%~18%` | `8%~12%` | `6%~10%` | 热身教学，允许轻微失误 |
| `1-2` | `18%~24%` | `12%~18%` | `8%~14%` | 学习第一轮升级和节奏滚动 |
| `1-3` | `30%~38%` | `18%~24%` | `12%~18%` | 首个软卡点，开始显性提示早期成长 |
| `2-1` | `24%~32%` | `16%~22%` | `12%~18%` | 新手包价值延续，但仍可白嫖通过 |
| `2-2` | `32%~40%` | `22%~28%` | `14%~20%` | 从“只升卡”过渡到“卡牌 + 研究” |
| `2-3` | `38%~46%` | `28%~34%` | `18%~24%` | 中期首领墙，开始验证研究和资源经营 |
| `3-1` | `30%~38%` | `22%~28%` | `16%~22%` | 熟练玩家段，要求构筑和研究都成型 |
| `3-2` | `38%~46%` | `28%~34%` | `20%~26%` | 为 `3-3` Boss 墙做预热和构筑切换 |
| `3-3` | `45%~55%` | `34%~42%` | `22%~28%` | 后段首领卡点，应明显感到“花钱后更可控” |
| `4-1` | `40%~48%` | `30%~38%` | `22%~28%` | 付费后不该平推，仍需合理切 Boss 构筑 |
| `4-2` | `50%~60%` | `38%~46%` | `26%~34%` | 终盘成长墙，要求长线资源闭环 |
| `4-3` | `58%~68%` | `44%~52%` | `30%~38%` | 终局 Boss 点，验证整套成长系统的上限 |

### 观测方式

- 优先看**首尝失败率**，判断玩家是否明确感知到了“这里是卡点”。
- 同时看**3 次内通关率**，避免把卡点做成纯数值硬墙。
- 重点盯 `1-3`、`2-3`、`3-3` 三个转化节点，确认失败率、留存和付费转化是否同步变化。

## 10. 关键卡点的补齐来源与通过路径

上面的总表会暴露一个很重要的事实：

- `Cipher Match` 的中后段不是靠章节奖励单独闭环的
- 真正的推进闭环应该由：**章节奖励 + 任务 + 免费赛季 + 每日补给 + 对位礼包**
- 这正好对应我们希望玩家感受到的成长节奏：
  `白嫖靠经营和时间补齐`、`小氪买 1~3 天追赶时间`、`中氪买当前卡点真正缺的那块增强`

### 关键卡点补齐池

| 卡点 | 白嫖保守缺口 `C/B/D` | 活跃免费补齐池 `C/B/D` | 白嫖通过路径 | 小氪通过路径 | 中氪通过路径 |
| --- | --- | --- | --- | --- | --- |
| `1-3` | `+944/+117/+73` | `+2040/+160/+22` | `R2 + S1` 即可稳过；这里是升级教学，不应做成硬收费墙 | `starterPack` 直接把余量拉高到 `+3544/+337/+145`，作用是减少早期追赶时间 | 这里不建议强行推中氪，价值点还没到 |
| `2-3` | `-2018/+48/+66` | `+4320/+336/+60` | 吃满 `m1~m6`、免费赛季 `f1~f3`、约 `2` 天每日补给后，可把 `R5 + S4 + P2` 补齐 | `starterPack` 已足够把保守模型拉回正余量，适合“省 2~3 天”而不是“跳章节” | `seasonPass` 让这段直接进入宽松区间，且真实效果还额外包含研究打折与能量循环 |

### 这一轮数值结论

1. **`1-3` 与 `2-3` 的礼包对位是成立的**
   `starterPack` 解决第一波早期资源和开局稳定性，`seasonPass` 解决中期研究与循环压力。

2. **`3-3` 之后不能再把“小氪”定义成只买首充包**
   如果希望后段小氪也有明确“我花这单是值的”感受，那么更合理的终盘小氪口径应是：
   `starterPack + seasonPass`

3. **`breakerVault` 的价值不应该只看即时资源**
   它真正解决的是：
   - Boss 反制压力
   - 后段收益偏低导致的拖关感
   - 中氪玩家在 `3-3 / 4-x` 的体验断层

4. **`4-2 / 4-3` 保持长线留存定位是合理的**
   只要白嫖仍有明确补齐路径，中氪也不是“一键平推”，这两个关口就能同时承担：
   - 长线留存
   - 终盘资源经营
   - 高价值付费转化
