> 更新时间：2026-04-24  
> 对应提交：`104191d` 之后的当前工作区

## 1. 建模前提

这份表按**保守模型**计算：

- 只计入：
  - 初始资源
  - 章节通关奖励
  - 章节首通奖励
- **不计入**：
  - 任务奖励
  - 赛季奖励
  - 每日免费补给
  - 商店追赶包

所以文档里的白嫖路径是“保守下限”，真实玩家因为会拿到任务、赛季、每日补给，实际可支配资源通常会**更宽松一档**。

### 初始资源

| 资源 | 数值 |
| --- | ---: |
| 金币 `Credits` | 1680 |
| 密钥位 `Key Bits` | 180 |
| 密码尘 `Cipher Dust` | 56 |

### 默认起始构筑

- 队长：`glintFox`
- 技能：`gridBurst`
- 模块：`prismTap` + `hardPatch`
- 起始构筑战力：`260`

### 核心构筑轮次定义

文档中的“**1 轮核心构筑**”指：

- 当前队长 +1 级
- 当前技能 +1 级
- 当前两张模块各 +1 级

当前版本下：

- 每轮核心构筑固定提升原始战力：`+76`
- 研究 `stabilityMesh` 每级再给整套构筑 `+4%`

## 5. 当前卡点设计

### A. `1-3` 首个软卡点

- **目标体验**：让玩家在第一个首领前学会：
  - 升当前主力构筑
  - 理解护盾目标
  - 认识“首补免费”和开局资源的重要性

#### 白嫖通过路径

- 推荐状态：
  - `2` 轮核心构筑
  - 或 `1` 轮核心构筑 + `stabilityMesh` `L2-L3`
- 资源参考：
  - `2` 轮核心构筑累计：`1407 / 163`
  - 玩家到 `1-2` 后保守资源：`2571 / 280 / 91`
- 结论：
  - **白嫖可稳过**
  - 这里不是强收费墙，而是第一波“升级教学”

#### 小氪通过路径

- 推荐购买：`starterPack`
- 对应增强：
  - 立即给 `2600 Credits / 220 Bits / 72 Dust`
  - 永久 `+1` 每日免费局
  - 每局开局额外能量更高
- 价值对位：
  - 约等于 **2 轮核心构筑 + 早期研究**
  - 直接覆盖 `1-3 / 2-1`

#### 中氪通过路径

- 不建议在这个点就上中氪
- 更合理的中氪入口是 `2-2 / 2-3`

---

### B. `2-2 / 2-3` 中期研究卡点

- **目标体验**：逼玩家从“只升卡”过渡到“卡牌 + 研究 + 商店补给”
- `2-3` 是第一道真正的中期资源压力点

#### 白嫖通过路径

- 推荐状态：
  - `4-5` 轮核心构筑
  - `stabilityMesh L4-L5`
  - `pulseBattery L2`
  - 领取任务 / 每日补给 / 赛季免费档位
- 保守资源参考：
  - 到 `2-2` 后保守资源：`5346 / 596 / 195`
  - `4` 轮核心构筑累计：`3812 / 396`
  - `stabilityMesh L4` 累计：`1399 / 102`
  - `pulseBattery L2` 累计：`371 / 27`
- 结论：
  - 单靠“只升卡”会开始发紧
  - 但白嫖只要把任务、每日免费补给和免费赛季领满，仍能过 `2-3`

#### 小氪通过路径

- 推荐购买：`starterPack`
- 通过方式：
  - 用礼包资源把核心构筑提到 `4-5` 轮
  - 早期研究补到 `stabilityMesh L4`
  - 靠永久开局增益稳过 `2-3`
- 设计意图：
  - 让小氪感受到“**省 2-3 天追赶时间**”，而不是直接跳过整个第二章

#### 中氪通过路径

- 推荐购买：`seasonPass`
- 对应增强：
  - 立即给 `4200 Credits / 320 Bits / 160 Dust / 180 Season XP`
  - 解锁高级赛季轨道
  - 研究价格额外 `-12%`
  - 第二章开始额外给开局能量
  - 最大能量额外 `+10`
- 价值对位：
  - 约等于 **3 轮核心构筑 + 稳定网格 L4**
  - 是专门对 `2-2 / 2-3` 研究卡点设计的中氪解法

---

### C. `3-3` 后段首领卡点

- **目标体验**：从“数值够不够”升级成“数值 + Boss 对策是否成型”
- 这里开始要求：
  - 构筑轮次更高
  - `stabilityMesh` 不再能缺席
  - 需要切到更适合首领的 Boss 构筑

#### 白嫖通过路径

- 推荐状态：
  - `7-8` 轮核心构筑
  - `stabilityMesh L5-L6`
  - `pulseBattery L3-L4`
  - 构筑换成 Boss 取向：
    - 队长优先 `novaEcho` / `wardenNine`
    - 模块优先 `traceMine` / `burstIndex`
    - 技能优先 `stasisField` 或 `colorHack`
- 保守资源参考：
  - 到 `3-2` 后保守资源：`11141 / 1253 / 413`
  - `7` 轮核心构筑累计：`10979 / 951`
  - `8` 轮核心构筑累计：`14976 / 1214`
- 结论：
  - 白嫖能过，但需要把任务 / 免费赛季 / 每日补给几乎都吃满
  - 这是“熟练玩家区间”的真正分水岭

#### 小氪通过路径

- 推荐购买：`starterPack + seasonPass`
- 通过方式：
  - `starterPack` 负责早期抢节奏
  - `seasonPass` 负责中期研究折扣 + 额外资源 + 高级轨道补给
- 设计效果：
  - 小氪玩家能在 `3-3` 前把资源差拉开一整轮
  - 但仍然需要切 Boss 构筑，不能无脑平推

#### 中氪通过路径

- 推荐购买：`breakerVault`
- 对应增强：
  - 立即给 `18000 Credits / 1350 Bits / 560 Dust`
  - 永久提升后段章节收益
  - 永久减轻首领反制
- 价值对位：
  - 约等于 **8 轮核心构筑 + 一组中高阶研究**
  - 是针对 `3-3 / 4-x` Boss 墙的专门付费解

---

### D. `4-2 / 4-3` 终盘卡点

- **目标体验**：必须完成“成长闭环”
- 这里不再只是等级门槛，而是：
  - 高轮次核心构筑
  - 中高等级研究
  - Boss 专项增益
  - 长周期资源经营

#### 白嫖通过路径

- 推荐状态：
  - `9-10` 轮核心构筑
  - `stabilityMesh L6+`
  - `pulseBattery L4+`
  - 免费赛季和任务尽量全吃
- 特点：
  - 可过，但时间成本很长
  - 这是终局长线留存区

#### 小氪通过路径

- 推荐购买：
  - `starterPack`
  - `seasonPass`
- 通过方式：
  - 保持日常经营，把 `4-2 / 4-3` 当作 1~2 周的终盘目标

#### 中氪通过路径

- 推荐购买：
  - `starterPack`
  - `seasonPass`
  - `breakerVault`
- 通过方式：
  - `breakerVault` 直接把后段 Boss 战从“硬拖”改成“可控”
  - 让中氪玩家感到“这单花得值”，但仍要完成构筑与研究

## 6. 三档付费与卡点映射

| 档位 | 推荐触发点 | 资源价值 | 永久增益 | 解决的核心问题 |
| --- | --- | --- | --- | --- |
| `starterPack` | `1-3 / 2-1` | 约等于 2 轮核心构筑 + 早期研究 | 每日免费局 +1、开局更稳 | 解决第一波升级焦虑 |
| `seasonPass` | `2-2 / 2-3` | 约等于 3 轮核心构筑 + 中期研究包 | 研究更便宜、能量循环更顺 | 解决中期研究卡点 |
| `breakerVault` | `3-3 / 4-x` | 约等于 8 轮核心构筑 + 高阶研究包 | 首领反制减轻、后段收益提高 | 解决后段 Boss 墙 |

## 7. 设计原则总结

当前版本的数值逻辑不是“硬性逼付费”，而是：

1. **白嫖能过**  
   只要任务、每日、免费赛季、研究和构筑都跟上，就能推进。

2. **小氪买时间**  
   小氪不直接跳章节，而是缩短 1~3 天的资源追赶周期。

3. **中氪买卡点解决方案**  
   中氪不是只买资源，而是买“当前卡点真正缺的那部分增强”：
   - 早期买开局稳定
   - 中期买研究效率
   - 后期买 Boss 对策

4. **高阶卡点必须保留操作价值**  
   就算买了 `breakerVault`，玩家也仍然需要切 Boss 构筑、合理放技能、先打护盾再打核心目标。

如果后续还要继续往下调，我建议下一轮优先看两件事：

- `2-3` 的真实失败率是否落在 `25%~40%`
- `3-3` 的付费前后通关率差是否控制在“明显感知到变强，但不会一键碾压”
